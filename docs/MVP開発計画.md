# ステップバイステップタスク管理アプリ - MVP開発計画

## MVP版の仕様（無料版のみ）

### コア機能に絞る
1. **タスク入力と5ステップ分割**
   - 1日3回までのAI分割制限
   - シンプルな入力画面

2. **タスク実行**
   - 1ステップずつ表示
   - 完了ボタンのみ
   - 進捗インジケーター（1/5, 2/5...）

3. **基本的なデータ保存**
   - ローカルストレージのみ
   - 現在のタスクの状態保持

### MVP版で実装しない機能
- ❌ ユーザーアカウント
- ❌ クラウド同期
- ❌ 詳細な統計
- ❌ カスタマイズ機能
- ❌ テンプレート

## 技術選定（シンプルに）

### モバイルアプリ開発
```
React Native（推奨）
- 理由：両OS対応、開発速度重視
- Expo使用でさらに高速開発
```

### AI統合
```
OpenAI API
- GPT-3.5-turbo（コスト効率）
- シンプルなプロンプトで実装
```

### データ管理
```
AsyncStorage
- React Nativeの標準
- JSONでシンプルに保存
```

## リスクと対策

### 技術的リスク
- **API制限**: レート制限の実装

### ビジネスリスク
- **競合出現**: 独自性の強化
- **低継続率**: UI/UX改善

## まとめ

MVP版は「シンプルさ」を最優先に：
- 1つのことをしっかりやる
- 余計な機能は入れない
- まずはリリースして反応を見る

このアプローチで、最速でユーザーの手に届けることを目指します。
