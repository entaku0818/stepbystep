One Task Steps - приложение для управления задачами, использующее ИИ для автоматического разделения задач на 5 выполнимых шагов с фокусом на одной задаче.

Основные функции:
- Разделение задач с помощью ИИ (Gemini API)
- Только одна задача за раз
- Последовательное выполнение: нельзя пропускать шаги
- Показывает только текущий шаг
- Отслеживание прогресса и анимации завершения

Особенности:
- Простой ввод текста с авто-разделением ИИ
- Можно зарегистрировать только одну задачу
- Минималистичный UI для поддержки концентрации
- 3 использования ИИ в день (бесплатная версия)

Целевые пользователи:
Идеально для людей с СДВГ или проблемами концентрации.

Простой дизайн для стабильного выполнения задач.
