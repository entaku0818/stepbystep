One Task Steps é uma app de gestão de tarefas que usa IA para dividir automaticamente tarefas em 5 passos executáveis, impondo foco numa única tarefa.

Funcionalidades principais:
- Divisão de tarefas com IA (usando Gemini API)
- Tarefa única obrigatória: Apenas uma tarefa de cada vez
- Execução sequencial: Não é possível saltar passos
- Mostra apenas o passo atual
- Acompanhamento de progresso e animações de conclusão

Destaques:
- Entrada de texto simples com divisão automática por IA
- Apenas uma tarefa pode ser registada
- UI mínima para apoiar o foco
- 3 usos de IA por dia (versão gratuita)

Utilizadores-alvo:
Perfeito para pessoas com TDAH ou dificuldades de concentração.

Design simples para execução constante de tarefas.
